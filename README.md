# 🚀 CodeSage AI

An AI-powered code review platform that analyzes source code, detects potential issues, evaluates code quality, estimates time and space complexity, and generates improved code suggestions using Large Language Models.

---

## ✨ Features

- 🤖 AI-powered code review
- ⭐ Overall code quality score
- 📊 Big-O Time & Space Complexity Analysis
- 🐞 Potential Bug Detection
- 💻 AI-generated Improved Code
- 📄 PDF Report Generation
- 📜 Review History
- 🌙 Dark / Light Theme
- 🧠 Monaco Code Editor
- 🌐 Multi-language support

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Gaurav24122006/CodeSage-AI.git
```

### Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn main:app --reload
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
OPENROUTER_API_KEY=your_api_key
OPENROUTER_MODEL=deepseek/deepseek-chat-v3-0324:free
```
